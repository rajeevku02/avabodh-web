Scratchpad

The Scratchpad will convert android tablet into a drawing pad which can be used to draw images and taking handwriting notes.

Features:
  • draw/erase using touch.
  • draw smooth curve. The Scratchpad will automatically convert the hand drawing stroke into a smooth curve if smoothing is enabled.
  • select pen width and color.
  • undo/redo actions.
  • export drawings as image so that the drawing can be viewed in Gallery app.
  • share drawings to other users.


Get scratchpad from here: https://play.google.com/store/apps/details?id=com.avabodh.scratchpad
